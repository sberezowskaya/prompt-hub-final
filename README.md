Описание проекта

Prompt Hub — это веб-приложение для создания, хранения и обмена промптами для ИИ-инструментов (ChatGPT, GigaChat, Midjourney и др.). Пользователи могут изучать техники промпт-инжиниринга, создавать шаблоны с подсветкой синтаксиса, добавлять в избранное и делиться публично.

Функциональные возможности

Обязательные

Просмотр исследований и техник промпт-инжиниринга
Редактор промптов с цветной подсветкой синтаксиса
Готовые шаблоны с заготовленными примерами
Панель быстрых действий (вставка тегов, переменных, JSON)
Публичный каталог с поиском и фильтрацией по инструменту и сфере
Вход, регистрация, восстановление пароля
Личный кабинет с редактированием профиля
Создание собственных шаблонов
Избранное (сохранение чужих промптов)
Дополнительные (повышенная сложность)

Модерация шаблонов (статусы: приватный, на модерации, опубликован)
Система оценки (лайки)
База знаний со статьями и фильтром
Технические

Поддержка доступности (aria-label)
Lighthouse аудит (зелёные показатели)
Технологический стек Фронтенд: Next.js 14, TypeScript, Tailwind CSS Бэкенд: Mock API (Next.js API routes)

Подсветка синтаксиса: react-syntax-highlighter

Тесты: Vitest + Testing Library

Поддерживаемые элементы в редакторе

Заголовки: ## Heading
Разделители: ---
Стрелки потока: →
XML-теги:
Переменные: {{var}}
Акценты: CAPS
MetaGlyph: ∈ ∪ → ⊕
Инлайн-код: code
JSON: "key": "value"
Декораторы: +++Format
